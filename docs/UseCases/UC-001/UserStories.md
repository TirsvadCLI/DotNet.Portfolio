# UC-001: Application user ahtentication

## User Story

**As** a user of the TirsvadWeb Portfolio,  
**I want** to log in with my credentials,  
**So that** I can access personalized features and manage my portfolio securely.

## Acceptance Criteria

- The user can enter a username and password.
- The system validates the credentials.
- If valid, the user is granted access to their portfolio.
- If invalid, an error message is displayed.

## Notes

- This use case is for authentication only; registration and password recovery are separate use cases.
